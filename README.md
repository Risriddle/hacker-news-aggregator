
# Hacker News Aggregator

Full-stack MERN application that automatically scrapes and stores top Hacker News stories, featuring JWT authentication, 
bookmarking functionality, REST APIs, and MongoDB persistence.


## Features

- JWT auth
- Web Scraping
- REST APIs



## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express



## Installation

Install scraper with npm

```bash
  npm install scraper
  cd scraper
```
    
## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`MONGO_DB_URL`

`JWT_SECRET`
